In-Field Labels jQuery Plugin

License: Same as jQuery (Dual GPL + MIT)

Author: Doug Neiner

Download: InFieldLabels.zip

Source: GitHub

Size: Under 1KB Minified and gzipped

Overview:
This is a simple plugin that turns properly formatted HTML forms into forms with in-field label support. Labels fade when the field is focussed and disappear when text entry begins. Clearing a field and leaving brings back the label.

Browser Support:
IE6+, WebKit Browsers (Safari, Chrome), Firefox 2+ * IE6 requires a background-color be set on the label to match the background of the field.

Know Issues:
Browser Auto-Complete can cause problems in hiding the labels. Less of an issue for Login Boxes, but much more of an issue with comment forms.

Use

HTML:
All you need is an input element.

  <input type="text" name="some_field">

CSS:
The overlaying labels will be appended inside the end of the body tag, so make sure they have the correct style applied. All you need to do is set the same text style that the input elements themselves have. Normally something like this: 

  label.infieldlabel {
    font-size: 11px;
    font-weight: normal;
  }

Javascript:

  $(function(){
    $(":text[name='some_field']").
      inFieldLabels("Some Field Label Text!");
  });

Options

Three options can be passed along with the method or set ahead of time for all inFieldLabel controls.

To set them ahead of time, use the following syntax:


  $.inFieldLabels.defaultOptions.optionName = "";

To pass them at call time, use the following syntax:


  $("label").inFieldLabels({ optionName:value });

fadeOpacity: Value between 0.1 and 1.0.
When a field is focussed, its label is animated to this opacity. Defaults to 0.5

fadeDuration: Time in milliseconds.
When an animation occurs, it uses this setting to determine duration. Defaults to 300

text: Label text.
You can pass in the label text as an option if you need to set other options at the same time, otherwise the string by itself will work fine.
